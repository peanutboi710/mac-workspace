# TIL: 2024-05-22 JavaScript Core Concepts

## 1. 반복문 (Loops)
- `for`, `while`: 기본 반복문.
- `for...in`: 객체의 키(Key) 순회.
- `for...of`: 배열의 값(Value) 순회.

## 2. 호이스팅 및 스코프 (Hoisting & Scope)
- `var`: 함수 스코프, 호이스팅 시 undefined 초기화.
- `let`, `const`: 블록 스코프, TDZ 존재.
- 함수 선언문은 전체가 호이스팅됨.

## 3. 함수 심화 (Advanced Functions)
- **IIFE**: `(function() { ... })();`
- **Arrow Function**: `const add = (a, b) => a + b;`
- **Rest Parameter**: `function sum(...args) { ... }`

## 4. 구조 분해 할당 (Destructuring)
```javascript
const obj = { name: "James", age: 20 };
const { name, age } = obj; // 객체 분해

const arr = [1, 2, 3];
const [first, ...rest] = arr; // 배열 분해 및 Rest
```

## 5. 클로저 (Closure)
- 외부 함수의 변수를 참조하는 내부 함수가 반환될 때, 그 환경이 유지되는 현상.
- 프라이빗 변수 구현 등에 활용.

## 6. 주요 메서드
### Array
- `push` / `pop`: 끝 요소 조작.
- `unshift` / `shift`: 앞 요소 조작.
- `splice(start, count)`: 요소 삭제 및 추출.

### Object
- `Object.keys(obj)`: 키 배열 반환.
- `Object.values(obj)`: 값 배열 반환.
- `Object.entries(obj)`: [키, 값] 쌍 배열 반환.

## 오늘 느낀 점
JavaScript의 유연함은 강력하지만, 호이스팅이나 클로저 같은 개념을 정확히 이해하지 못하면 예상치 못한 버그가 발생할 수 있음을 깨달았다. 특히 `const`와 `let`을 활용한 블록 스코프 관리에 익숙해져야겠다.
