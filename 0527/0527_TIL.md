## 2. Arctic Breeze 디자인 시스템 및 티라노사우루스 웹사이트 구현
* 개요: 극한의 더위를 मनोवैज्ञानिक(심리학적)으로 냉각하는 "Thermal Relief" 콘셉트의 단일 페이지 웹사이트를 제작했습니다.
* 적용 디자인 사양 (Arctic Breeze):
  * 색상: 주조색(빙하 청록색), 보조색(쿨 틸), 배경색(프로스트 화이트) 등의 한색 위주 배색을 고수했습니다.
  * 레이아웃: 백드롭 필터(blur 20px)를 동반한 반투명 유리 질감(Glassmorphism) 카드를 배치하여 빙판의 속성을 유도했습니다.
  * 타이포그래피: Plus Jakarta Sans(헤드라인)와 Hanken Grotesk(본문)의 어우러짐을 활용했습니다.
* 인터랙션 기능:
  * 티라노사우루스의 백악기 원시 형태부터 현대적 사이버네틱 아크틱 사바나 형태까지의 진화사를 타임라인 형식으로 표현했습니다.
  * 지구의 온도를 45도에서 마이너스 15도까지 낮추는 아이스 브레스 시뮬레이터를 자바스크립트로 구현했으며, 클릭 좌표를 계산하여 화면에 미세 눈꽃 입자가 생성되고 소멸되는 파티클 애니메이션을 적용했습니다.

## 3. Verset 디자인 시스템 및 윌리엄의 사랑의 오작교 홈페이지 구현
* 개요: 미니멀리즘과 서정성에 기반을 둔 '고요함(Quietude)' 브랜드 시스템에 맞춰 오작교를 시각화한 포트폴리오 사이트를 제작했습니다.
* 적용 디자인 사양 (Verset):
  * 색상: 고품질 편지지 촉감을 재현하기 위해 웜 톤의 미색 양지 배경(#fcf9f8)과 매끄러운 숯빛 기화 텍스트(#1b1c1c), 그리고 세이지 그린과 더스티 로즈를 포인트로 활용했습니다.
  * 타이포그래피: 고전적 도서의 느낌을 재현하기 위해 Literata 세리프 체로 문학 본문을 인쇄하고, DM Sans 산세리프 체로 조작 UI 라벨을 정렬하여 시문과 인터페이스 간의 위계를 엄격히 조율했습니다.
  * 규칙: 화면 내 가독성을 저해하는 어떠한 불필요한 이미지 데코레이션이나 이모지 사용을 엄격히 배제했습니다.
* 인터랙션 기능:
  * 은하수의 양단에 위치한 견우성와 직녀성 별자리 요소를 수평 조절 슬라이더(Input Range) 수치에 따라 중심부(45%)로 동적 이동시켰습니다.
  * 두 별자리가 맞닿는 임계치(100%)에 도달할 때 은하수가 연결되며 사랑의 메시지가 부드럽게 출현하는 물리적 상호작용 메커니즘을 구성했습니다.

## 5. GitHub 보안 규칙 우회 해결 및 Git 히스토리 관리
* 문제 상황: 깃 푸시 과정에서 사용자가 opencode.json에 등록했던 구글 서비스 계정 API 키가 GitHub의 비밀정보 유출 탐지 시스템(Push Protection Rule)에 걸려 push가 전면 반려되는 장애를 겪었습니다.
* 해결 절차 및 학습 포인트:
  1. 기밀 데이터 보호: opencode.json 내의 민감한 실측 API 키 값을 원격에 저장하지 않기 위해 안전한 일반 플레이스홀더 문자열(YOUR_GCP_API_KEY)로 치환했습니다.
  2. 로컬 커밋 청소 (Git Soft Reset): 이미 이전 로컬 커밋 내역(9fd2dab0...)에 기밀 정보 유출 흔적이 남아 있으므로, 이를 완전히 소거하기 위해 `git reset --soft` 명령으로 해당 커밋 이전 단계로 타임라인을 복원했습니다.
  3. 형상 재구성 및 전송: 안전하게 수정된 opencode.json을 다시 스테이징 영역에 추가(`git add`)한 후 새로운 클린 커밋을 만들어 푸시함으로써, 민감한 키의 외부 노출을 근본적으로 차단하고 깃허브 보안 필터를 성공적으로 우회하여 전체 이력을 무사히 푸시했습니다.
